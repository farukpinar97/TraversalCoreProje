# :file_folder: Traversal Rezervasyon Core Projesi
Bu proje **Murat Yücedağ** eğitmenliğinde **"Traversal Rezervasyon Mini Core Projesi"** youtube video serisi takip edilerek geliştirilmiş bir seyahat web projesidir.Proje **Admin Paneli**, **Üye Paneli** ve **Vitrin** olmak üzere üç ana bölümden oluşmaktadır.
## :clipboard: Proje Hakkında
**Traversal Rezervasyon Core** projesi Asp.NET CORE 8 ile Entity Framework Code First yaklaşımıyla N Katmanlı mimari kullanılarak geliştirilmiştir. Projede Business,Entity,DataAccess ve Presentation olmak üzere 4 temel katman bulunmaktadır.
Projenin vitrin sayfasında tur rotaları ve bu rotaların detayları listelenmektedir , kullanıcılar iletişim sayfası üzerinden mail gönderebilmektedir. Rota detay sayfasında rotalar hakkında yorum bırakabilir ve rehberleri görüntüleyebilirler.
Sisteme kayıt olan misafirler , üye paneli üzerinden yeni rezervasyon oluşturabilir , aktif , geçmiş ve onay bekleyen rezervasyonlarını görüntüleyebilirler. Menü dilini İngilizce ve Türkçe olarak değiştirebilirler. Şifresini unutan kullanıcılar
şifremi unuttum kısmında gönderilen link yardımıyla şifrelerini yenileyebilirler. Admin panelinde ise web sitesi yönetimi ve kullanıcı yönetimi sağlanmaktadır. Admin yeni rotalar , rehberler , roller
hakkında ekleme , silme , güncelleme işlemi yapabilmektedir.Rotaların Excel / PDF raporlarını oluşturabilirler.
## :dart: Kullanılan Teknolojiler
- **HTML , CSS , BOOTSTRAP**
- **Microsoft SQL Server**
- **ASP.NET CORE 8**
- **N Katmanlı Mimari**
- **Entity Framework Code First**
- **ASP.NET MVC Authentication , Authorization**
- **Fluent Validation**
- **SignalR**
- **CQRS ve UnitOfWork Design Pattern**
- **Localization**
- **Booking API**
